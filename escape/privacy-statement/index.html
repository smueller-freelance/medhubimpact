Privacy Policy

This app is an independent, two-person project. Protecting your privacy is important. This policy explains what data is collected, why it is collected, and how it is used.

Data Controller

The data controller is the app developer.
No external data protection officer is appointed, as this is a small project.

Data We Collect

The app collects only the data strictly necessary for its core functionality.

User Data

  • Device ID (used to distinguish users anonymously)
  • Nickname (chosen by the user)
  • App opening timestamps
  • Images viewed (references to image identifiers)

Image Interaction Data

  • Votes on images
  • Vote timestamps
  • Number of views per image
  • Aggregated statistics such as total and average votes

Daily Aggregated Data

  • Date
  • Total number of votes per day
  • Number of users per day
  • References to images shown on that day

Submitted Content

  • Nickname
  • Submitted image file name

Purpose of Data Processing

Data is processed solely to:

  • Enable daily image sessions
  • Count and display votes
  • Show aggregated results (e.g. top laughs, shared votes)
  • Prevent abuse
  • Review and publish user-submitted images

Legal Basis

Processing is based on Article 6(1)(b) GDPR (performance of the service) and Article 6(1)(f) GDPR (legitimate interest in operating and improving the app).

Data Sharing

No personal data is sold or shared with third parties.
No advertising networks, trackers, or analytics services are used.

Data Storage

Data is stored only as long as necessary for the app’s functionality. Aggregated statistics may be stored without personal reference.

User Rights

Under the GDPR, you have the right to:

  • Access your data
  • Request correction or deletion
  • Object to processing
  • Request data portability

Requests can be made by contacting the app developer.

Children

This app is not intended for children under the age of 13.

Changes

This policy may be updated if the app’s functionality changes.

Contact Us

If you have questions about this privacy policy, please contact us at: support@medhubimpact.com
